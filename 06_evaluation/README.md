# 06_evaluation — Avaliação

Reúne os resultados quantitativos e qualitativos dos dois pipelines do projeto.

## Conteúdo

| Pasta | Escopo |
|---|---|
| `classification/` | Métricas do Pipeline 1 (classificação de sentimento) |
| `recommendation/` | Métricas do Pipeline 2 (recomendação re-ranqueada) |
| `qualitative/` | Avaliação manual e concordância entre avaliadores (Kappa de Cohen) |

> A avaliação qualitativa em `qualitative/` cobre **ambos** os pipelines e é conduzida de forma independente por cada integrante do grupo.
